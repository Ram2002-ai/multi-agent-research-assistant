# Roadmap

## Delivered

- Live SSE and WebSocket agent execution
- Durable timeline, searchable logs, retries, and checkpoints
- Report history, saving, source credibility scores, and citation styles
- Persistent local vector memory with semantic retrieval and reranking
- PDF, DOCX, HTML, Markdown, JSON, CSV, and ZIP export
- Analytics, Mermaid definitions, and interactive graph data
- Multi-provider model catalog and runtime model selection
- Prompt version history and single-agent playground
- JWT, Google OAuth, GitHub OAuth, roles, and production auth switch
- SQLite development and PostgreSQL production repository
- Responsive React dashboard, dark mode, Docker, CI, and tests

## Next production increments

- Alembic schema migrations
- Redis/Celery or Temporal distributed execution
- S3-compatible artifact storage
- Provider-reported token and cost accounting
- Learned cross-encoder reranking
- Organization workspaces, invitations, and fine-grained RBAC
- Admin audit log and budget controls
- Prompt comparison experiments and evaluation datasets
- Playwright visual regression and end-to-end streaming tests
- Native PowerPoint report renderer
