## Step-by-Step Explanation (Teacher Agent)

**What is Artificial Intelligence?**

Artificial Intelligence (AI) refers to computer systems that can perform tasks that normally require human intelligence.

**How does it work?**

1. Data is collected (text, images, numbers)
2. An algorithm learns patterns from this data
3. The trained model makes predictions or decisions on new data

**Real-world example:**
When Netflix recommends a show you might like, it's using AI. The system has learned from millions of users' viewing patterns and predicts what you'll enjoy next — without anyone explicitly programming the rules.
